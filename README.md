# Modern Portfolio Website (React + Tailwind)

## Setup

1. `npm install`
2. `npm run dev`
3. Open the local URL shown in terminal (usually `http://localhost:5173`).

## Features

- Navbar with smooth scroll + active section indicator
- Hero section with photo, role, CTA buttons
- About section with resume download
- Skills section with icons
- Projects section with cards (GitHub + demo links)
- Contact form (validation + success state)
- Footer with GitHub/LinkedIn
- Responsive mobile/tablet/desktop
- CSS animations via Framer Motion

## Customize

- Edit text and links in `src/components/*`
- Replace profile image in `src/components/Hero.jsx` and `src/components/About.jsx`
- Add more skills to `src/components/Skills.jsx`
- Add projects in `src/components/Projects.jsx`
